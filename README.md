# chatbot-code-corrector-
A simple command-line chatbot built in Python using the OpenAI API.  Reads your API key securely from an environment variable (OPENAI_API_KEY).  Supports interactive conversation with GPT models (gpt-3.5-turbo or gpt-4o).  Clean and modular code structure for easy customization.  Safe exit commands: quit, exit, or bye.
